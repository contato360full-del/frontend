# Biotrato Saúde Integrativa — Landing (React + Tailwind)

Projeto mobile-first de landing page premium criado com Vite + React + TailwindCSS.

Instalação e execução:

1. Instale dependências:

```bash
npm install
```

2. Rodar em modo desenvolvimento:

```bash
npm run dev
```

3. Build de produção:

```bash
npm run build
```

Configurar WhatsApp:

Edite `src/config.js` e substitua `WA_PHONE` pelo número da clínica no formato internacional, por exemplo `+5511999999999`.

Observações:
- Os arquivos de imagem em `assets/images` são placeholders SVG — substitua por fotos otimizadas.
- Este projeto prioriza experiência móvel, design premium e conversão por WhatsApp.
